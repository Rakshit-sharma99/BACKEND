# Telegram Platform

This directory is reserved for the Telegram integration module.

## Planned Architecture

The Telegram platform will follow the same pattern as WhatsApp:

- `sessionManager.js` — Telegram Bot/client session lifecycle
- `messageHandler.js` — normalize Telegram messages
- `search.js` — search Telegram community data
- `storage/` — per-user database and context files
