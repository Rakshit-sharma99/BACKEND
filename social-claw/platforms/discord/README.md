# Discord Platform

This directory is reserved for the Discord integration module.

## Planned Architecture

The Discord platform will follow the same pattern as WhatsApp:

- `sessionManager.js` — Discord Bot session lifecycle
- `messageHandler.js` — normalize Discord messages
- `search.js` — search Discord community data
- `storage/` — per-user database and context files
